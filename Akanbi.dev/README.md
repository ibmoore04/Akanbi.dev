# Portfolio Website

This repo is now a plain static portfolio site with no framework or build-tool dependency.

## Project Structure

- `index.html`: Main page markup
- `styles.css`: Extracted site styles
- `script.js`: Extracted site interactions
- Media files: Local images used by the portfolio

## Usage

Open `index.html` directly in a browser, or serve the folder with any simple static server.

## Notes

- The old Vue files have already been removed.
- Unused Tailwind, PostCSS, and Vite setup was removed.
- The layout and behavior were preserved while splitting the large HTML file.
